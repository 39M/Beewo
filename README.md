# Beewo
2016 年，第一次参加 Game Jam 的作品。主题是 **黑化**。  
通过一个音乐节奏游戏的 6 次版本迭代，表现小团队逐渐向现实妥协的过程。

[视频演示](https://www.bilibili.com/video/BV1SN411X7MC/)
